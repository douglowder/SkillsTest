# SkillsTest

A single-screen Expo TV video player app demonstrating React Native TV, NativeWind v4, and `expo-video`.

## About this project

This project was generated by [Claude](https://www.claude.com/) using the [react-native-tvos/skills](https://github.com/react-native-tvos/skills) agent skills. The skills guided Claude in:

- Creating an Expo TV project from the `with-router-tv` template
- Configuring NativeWind v4 (Metro, Babel, Tailwind, CSS, TypeScript declarations)
- Adding `expo-video` and wiring up a TV-focus-aware video player screen

## Features

The app has a single screen with:

- A video player (`expo-video`) that plays a local sample video
- A blue progress bar directly under the video, updating every second
- Five TV-focusable control buttons built with NativeWind:
  - **Rewind** — jumps to the start of the video
  - **-10s / +10s** — seeks backward or forward 10 seconds
  - **Play / Pause** — toggles playback (label updates with state)
  - **Fullscreen** — enters native fullscreen mode
- TV focus navigation via `TVFocusGuideView`, with the Play button receiving initial focus
- The inline video view re-renders on `readyToPlay` so playback resumes cleanly after exiting fullscreen

## Tech stack

- **Expo SDK 55** with Expo Router
- **react-native-tvos** (TV fork of React Native)
- **@react-native-tvos/config-tv** Expo config plugin for TV builds
- **NativeWind v4** with Tailwind CSS
- **expo-video** for playback

## Running the app

```sh
yarn
yarn prebuild:tv      # Expo prebuild with TV modifications
yarn ios              # Build and run on Apple TV
yarn android          # Build and run on Android TV
```

For mobile, use `yarn prebuild` and the same `yarn ios` / `yarn android` commands.

## Project structure

- `src/app/_layout.tsx` — root layout, imports `global.css`
- `src/app/index.tsx` — the video player screen
- `src/global.css` — Tailwind directives
- `tailwind.config.js`, `babel.config.js`, `metro.config.js` — NativeWind v4 configuration
- `assets/bach-handel-corelli.mp4` — sample video
